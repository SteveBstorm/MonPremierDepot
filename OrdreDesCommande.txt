1. Après modification ou création d'un fichier
	git add .

2. Pour créer un point de sauvegarde 
	git commit -m "message"

3. Pour envoyer les dernières modifs en ligne
	git push -u origin master

Salut à tous !
Effectuer un clone d'un dépot distant
	git clone adresseDuDepot

Récupérer la nouvelle version d'un dépot distant
	git pull
